Molly, the Molecule Manipulator

 Home   Applet   How to use   Download 

The Molly project is a chemistry program using the Jmol Java interface. Its purpose is to allow chemistry students to discover and test symmetry operations on simple molecules.

Try it out on the applet page. Information about how to use it is also available.

System requirements:
  • Java must be installed on your system.
  • You must have an internet connection to load molecules.

Please contact James Vanderhyde for more information.
